# opencv检测斑马线


现在需要以


基于openCV设计一个程序由于识别斑马线，流程如下，每一步输出处理的图像结果方便调试参数
1. 灰度化
2. 平滑滤波
3. 高斯模糊
4. 阈值处理（设置灰度阈值，低于阈值部分被掩码掩盖）
5. canny边缘检测
6. hough变化检测直线
7. 统计直线数量超过阈值则认为识别到斑马线





## 总结类

1. 传统算法
   1. 基于阈值
   2. 基于区域提取
   3. 基于边缘检测
2. 分割方法
   1. 概率图模型
   2. 像素，超像素分类

像素或者图像域之上的条件随机场，传统方法还是依靠图像的一些浅层信息，而仅仅依靠浅层信息往往难以达到精细分割的要求
